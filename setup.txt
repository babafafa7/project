Mysql Docker :

> Download mysql and run it
$ docker run --name db-mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=password123 -d mysql

> List container running
$ docker ps

> List ALL container 
$ docker container ls -a

> Start Stop container
$ docker start|stop container_id

> Bash container
$ docker exec -it container_id bash

> Connect mysql in container & create database
$ mysql -uroot -p
$ create database dbproject;
$ use dbproject;
$ select * from tablename;
$ exit
